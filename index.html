About OneOS

OneOS is an Ubuntu-based Linux distribution, currently in version Alpha 0.1.8. This means that all icons and the splash screen now feature OneOS's looks. However, it still uses Ubuntu Focal Fossa's window manager.

There are currently no files in the archive directory. There will be a full Alpha 0.1.8 archive repository in due time.
